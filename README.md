# 🌸 Iris Dataset Exploratory Data Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Iris dataset using Python.

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset

- 150 observations
- 6 variables
- No missing values

## Analysis Performed

- Data Loading
- Data Inspection
- Missing Value Analysis
- Descriptive Statistics
- Histogram
- Scatter Plot
- Correlation Matrix
